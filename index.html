Tipos de Saxofones

Saxofon Soprano.

El saxofon Soprano recto esta hecho para el lado de jazz.

Saxofon Soprano Curco.

Es lo mismo que el saxofon soprano pero curvo, hecho para jazz y clasico.

Saxofon Alto.

Saxofon mas comun para inciar en este mundo.

Saxofon Tenor.

Saxofon mas grande que el saxofon alto, hecho para jazz y clasico.

Saxofon Baritono.

Saxofon mas grande que el saxofon tenor, hecho para classico y salsa.

Saxofon Contrabajo.

Saxofon mas grande que el Saxofon Baritono, no tan comun.
